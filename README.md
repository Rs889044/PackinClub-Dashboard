
# 🌿 PackinClub — Invoice Management Dashboard

Manage customers, products, invoices, quotations, and orders in one place.

## Quick Start

1. Download this project → Click green **"Code"** button → **"Download ZIP"**
2. Unzip the folder anywhere on your computer
3. Double-click **`INSTALL_AND_RUN.bat`** (first time only)
4. After setup, use **`START_Dashboard.bat`** to launch daily

## Features

- 👤 Company Profile & Logo
- 👥 Customer Management
- 📦 Product Catalogue
- 🧾 Invoice Generation with PDF Download
- 📋 Quotations with Confirm-to-Order
- 📊 Orders with Excel/CSV Export

## Requirements

- Windows PC
- Python 3.10+ (installer guides you if missing)

## Data

All data is saved locally in `dashboard.db`. To backup, just copy that file.
